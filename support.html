빠른 시작 가이드

1단계: 카메라 디바이스 준비

사용하지 않는 스마트폰에 CCTV 카메라 앱을 설치하고 원하는 위치에 배치합니다.

2단계: 초대 코드 생성

카메라 앱에서 '초대 코드 생성' 버튼을 눌러 6자리 코드를 생성합니다.

3단계: 뷰어 앱 연결

iPhone에서 CCTV Viewer 앱을 열고 초대 코드를 입력하여 연결합니다.

4단계: 실시간 모니터링

연결된 디바이스를 선택하여 실시간 영상을 확인하세요.

자주 묻는 질문 (FAQ)

초대 코드를 잃어버렸어요

카메라 디바이스에서 다시 초대 코드를 생성할 수 있습니다. 카메라 앱의 설정 메뉴에서 '초대 코드 생성'을 선택하세요. 기존에 연결된 뷰어들은 영향을 받지 않습니다.

영상이 끊기거나 지연됩니다

다음 사항을 확인해주세요:

  • 카메라 디바이스와 뷰어 디바이스 모두 안정적인 Wi-Fi 또는 모바일 데이터에 연결되어 있는지 확인
  • 카메라 디바이스의 배터리가 충분한지 확인 (권장: 충전기 연결)
  • 다른 앱들이 네트워크를 많이 사용하고 있지 않은지 확인
  • 앱을 재시작하거나 연결을 다시 시도해보세요
푸시 알림이 오지 않아요

다음을 확인해주세요:

  • iPhone 설정 > 알림 > CCTV Viewer에서 알림이 허용되어 있는지 확인
  • 카메라 앱에서 소리 감지 기능이 활성화되어 있는지 확인
  • 소음 임계값이 너무 높게 설정되어 있지 않은지 확인 (기본값: 70dB)
  • 카메라 디바이스가 인터넷에 연결되어 있는지 확인
여러 대의 카메라를 사용할 수 있나요?

네, 가능합니다! 각 카메라 디바이스에서 생성된 초대 코드를 사용하여 여러 디바이스를 연결할 수 있습니다. 메인 화면에서 연결된 모든 디바이스 목록을 확인하고, 원하는 디바이스를 선택하여 모니터링하세요.

다른 사람과 카메라를 공유할 수 있나요?

네, 초대 코드를 공유하면 여러 명이 같은 카메라를 모니터링할 수 있습니다. 카메라 소유자가 생성한 초대 코드를 가족이나 동료에게 전달하면, 그들도 CCTV Viewer 앱에서 해당 코드를 입력하여 연결할 수 있습니다.

영상은 어디에 저장되나요?

CCTV Viewer는 실시간 스트리밍만 제공하며, 서버에 영상을 저장하지 않습니다. 모든 영상은 카메라 디바이스에서 뷰어로 직접 전송됩니다 (P2P). 스크린샷 기능을 사용하면 현재 화면이 iPhone의 사진 앱에 저장됩니다.

배터리 소모가 심한데 어떻게 하나요?

CCTV 카메라 디바이스는 지속적으로 영상을 송출하므로 배터리 소모가 많습니다. 장시간 사용 시 다음을 권장합니다:

  • 카메라 디바이스를 충전기에 연결하여 사용
  • 화면 밝기를 낮추거나 화면을 꺼둠
  • 불필요한 백그라운드 앱 종료
  • Wi-Fi 사용 (모바일 데이터보다 배터리 절약)
카메라 연결을 해제하려면 어떻게 하나요?

메인 화면에서 연결 해제하려는 디바이스를 왼쪽으로 스와이프하면 '연결 해제' 버튼이 나타납니다. 버튼을 눌러 연결을 해제할 수 있습니다. 연결을 해제해도 카메라 소유자가 다시 초대 코드를 생성하면 재연결할 수 있습니다.

소리 감지 임계값을 변경할 수 있나요?

네, 카메라 디바이스의 설정에서 소리 감지 임계값을 조정할 수 있습니다. 기본값은 70dB이며, 환경에 따라 60dB~90dB 사이로 조정하실 수 있습니다. 너무 낮게 설정하면 불필요한 알림이 많이 올 수 있으니 적절한 값을 찾아 설정하세요.

인터넷 연결 없이도 사용할 수 있나요?

아니요, CCTV Viewer는 실시간 영상 스트리밍을 위해 인터넷 연결이 필요합니다. 카메라 디바이스와 뷰어 디바이스 모두 Wi-Fi 또는 모바일 데이터에 연결되어 있어야 합니다. 같은 로컬 네트워크에 있어도 인증 및 연결 관리를 위해 인터넷 연결이 필요합니다.

  • 호환 운영체제 필요
  • 카메라 권한 필요
  • 마이크 권한 필요 (소리 감지)
  • 인터넷 연결 필수

문제 해결

연결이 안 되는 경우

  1. 초대 코드가 정확한지 확인 (대소문자 구분 없음)
  2. 카메라 디바이스가 켜져 있고 카메라 앱이 실행 중인지 확인
  3. 두 디바이스 모두 인터넷에 연결되어 있는지 확인
  4. Firebase 로그인이 정상적으로 되었는지 확인
  5. 앱을 완전히 종료했다가 다시 실행

영상 품질이 낮은 경우

  1. 인터넷 연결 속도 확인 (최소 5Mbps 권장)
  2. Wi-Fi 신호 강도 확인
  3. 카메라 렌즈가 깨끗한지 확인
  4. 카메라 디바이스의 성능 확인
  5. 다른 네트워크 사용 앱 종료

앱이 자주 종료되는 경우

  1. iPhone의 저장 공간 확인 (최소 1GB 여유 권장)
  2. iOS 업데이트 확인
  3. 앱 삭제 후 재설치
  4. iPhone 재시작
  5. 문제가 계속되면 고객 지원에 문의
