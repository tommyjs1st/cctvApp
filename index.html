주요 기능

📹

실시간 HD 영상

WebRTC 기반 저지연 실시간 스트리밍으로 1280×720 HD 화질을 제공합니다.

🔔

스마트 소리 감지

설정 가능한 소음 임계값으로 중요한 순간을 푸시 알림으로 즉시 받아보세요.

🔐

안전한 접근 제어

초대 코드 기반 보안 시스템으로 가족과 동료만 안전하게 공유합니다.

📸

화면 캡처

중요한 순간을 스크린샷으로 저장하고 증거 자료로 활용하세요.

👥

다중 디바이스

여러 CCTV 디바이스를 동시에 관리하고 쉽게 전환할 수 있습니다.

💰

비용 절감

별도 하드웨어 구매 없이 기존 스마트폰을 재활용하여 비용을 절감하세요.

사용 방법

1

카메라 앱 설치

여분의 스마트폰에 CCTV 카메라 앱을 설치합니다.

2

초대 코드 생성

카메라 앱에서 초대 코드를 생성합니다.

3

뷰어 앱 설치

iPhone에서 CCTV Viewer 앱을 다운로드합니다.

4

코드 입력

초대 코드를 입력하여 연결합니다.

5

실시간 확인

언제 어디서나 실시간 영상을 확인하세요!
